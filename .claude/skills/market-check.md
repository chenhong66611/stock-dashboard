# 行情分析 Skill

用户说"看看行情"时，按以下流程执行：

## 1. 查行情数据
- 使用 `node stock-quote.cjs sh510300 sh588000 sh560010` 获取实时价格
- 或通过 Tencent API 直接获取

## 2. 对比网格
记录当前价格与网格买入/卖出价的差距（百分比）：

| ETF | 买入① | 买入② | 买入③ | 卖出 |
|:----|:-----:|:-----:|:-----:|:---:|
| 沪深300 510300 | 4.54 | 4.35 | 4.15 | 4.99 |
| 科创50 588000 | 1.76 | 1.65 | 1.55 | 2.02 |
| 中证1000 560010 | 2.78 | 2.62 | 2.46 | 3.10 |

## 3. 搜新闻
- 搜索当前日期附近的市场新闻（A股大盘、ETF相关）
- 搜索可能影响科技/蓝筹的重大政策或事件

## 4. 分析
结合价格+新闻，给出：
- 今日走势判断
- 对网格触发的概率评估
- 明确的操作建议（买/卖/等）

## 5. 输出格式

```
📊 今日行情分析 — YYYY/MM/DD

🕐 当前价格：
  沪深300: xx.xxx (xx%)
  科创50:  xx.xxx (xx%)
  中证1000: xx.xxx (xx%)

📰 今日重点新闻：
  • xxx
  • xxx

🎯 网格触发情况：
  沪深300: 距买入① xx%
  科创50:  距买入① xx%
  中证1000: 距买入① xx%

💡 建议：
  [明确的操作意见]
```
